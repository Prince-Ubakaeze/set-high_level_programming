# High-Level Programming - Python

This repository contains projects and exercises focused on learning high-level programming principles, fundamentals of computer science, and practical scripting using Python.

## Directory Structure

Introduction to Python syntax, basic operations, and scripts execution.

## Requirements

*   Allowed editors: `vi`, `vim`, `emacs`, `VS Code`
*   All files interpreted/compiled on Ubuntu 20.04 LTS or 22.04 LTS using `python3` (version 3.8.x or newer)
*   All script files must be executable and end with a new line
